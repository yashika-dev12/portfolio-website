## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yashika-dev12/portfolio-website.git
```

Navigate to the project:

```bash
cd portfolio-website
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
